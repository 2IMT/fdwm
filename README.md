# fdwm

**fdwm** is a modified [dwm](https://dwm.suckless.org/), an extremely fast, small, and dynamic window manager for X.

## Requirements

- XLib
- Xinerama
- Freetype2
- Xft
- Fontconfig

## Installation

```bash
make clean install
```

## Configuration

The configuration of fdwm is done by modifying `src/config.h`
and (re)compiling the source code.
